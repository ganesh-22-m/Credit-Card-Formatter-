# 💳 Credit Card Formatter

A simple and interactive web application that formats credit card numbers in real-time using HTML, CSS, and JavaScript. The application automatically adds spacing between digits to improve readability and user experience.

---

## 🚀 Features

* 💡 Real-time credit card number formatting
* 🔢 Automatically adds space after every 4 digits
* ⌨️ Restricts non-numeric input
* 🎨 Clean and responsive UI design
* ⚡ Lightweight and fast

---

## 🧠 How It Works

The application uses JavaScript to:

* Remove non-numeric characters
* Group digits into blocks of 4
* Display formatted output dynamically

Example:

```id="ex01"
Input:  1234567812345678
Output: 1234 5678 1234 5678
```

👉 This formatting is commonly implemented using regex and string manipulation techniques in JavaScript. ([Peterbe][1])
